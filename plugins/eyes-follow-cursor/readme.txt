An interactive block that displays a pair of eyes that follow your cursor around the screen and occasionally wink at you.

== Description ==

The Eyes Follow Cursor block adds a delightful interactive element to your website. A pair of animated eyes will track the user's mouse movements across the screen, creating an engaging and playful experience. The eyes will also randomly wink at the cursor, adding personality to your site.

Perfect for:
* Personal blogs and portfolios
* Creative agency websites
* Entertainment and gaming sites
* Any site looking to add interactive personality

The block is fully responsive and works smoothly across all devices with pointer input.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/eyes-follow-cursor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the "Eyes Follow Cursor" block to any page or post using the block editor

== Frequently Asked Questions ==

= Does this work on mobile devices? =

The eyes will follow touch interactions on mobile devices. The winking animation works on all devices.

= Can I customize the appearance of the eyes? =

Currently, the block uses a default eye design. Future versions may include customization options.

= Does this affect page performance? =

The block is optimized for performance and uses requestAnimationFrame for smooth animations without impacting page load times.
