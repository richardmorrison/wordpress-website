TaskStreak is a calm, offline-first recurring task tracker with streak motivation and a quiet companion vibe.

== Description ==
TaskStreak creates a dedicated "My Tasks" page that renders a standalone app shell, so it feels like a separate app rather than a theme page.

Features:
* Recurring tasks with selected days
* Daily progress and streak tiers
* One grace miss per week per task
* Subtle undo after completing
* Badge history inside Settings
* Calm coach messages with a large rotating pool
* PWA caching for offline use

== Installation ==
1. Upload the plugin ZIP and activate
2. Visit the "My Tasks" page that was created

If your menu points to an older page, add the shortcode [taskstreak] to that page.

== Frequently Asked Questions ==
= Why do I still see my theme header and footer? =
Your menu likely points to a normal page using the shortcode fallback. Update your menu to the dedicated app page for the full app shell.
