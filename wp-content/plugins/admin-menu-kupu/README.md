# Admin Menu Kupu (Māori Labels)

**Admin Menu Kupu (Māori Labels)** overlays Māori translations in wp‑admin while preserving the original English (or site language) on hover. Includes per‑user toggles, a global default, minimal accessible tooltips, and an optional extension to overlay page titles/headers.

- Requires WordPress 6.0+, PHP 7.4+
- Version: 0.6.0
- Text Domain: `admin-menu-kupu`

## Highlights
- Safe overlay (does not change locale or core translations).
- Shows original label on hover via `title` and an accessible tooltip bubble.
- Per‑user enable/disable (user meta) + global default.
- Admin Bar "Kupu" quick toggle (AJAX; updates in place).
- Glossary JSON with in‑plugin editor + import/export.
- Optional H1/title overlay module (off by default).
- No audio; a commented stub is left for future extension.
